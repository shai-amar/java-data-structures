## Graph traversal
### Directed Graph
A graph G = (V,E) is directed if edge (x,y) ∈ E does NOT impliy that (y,x) is also in E
### Undirected graph
A graph G = (V,E) is undirected if edge (x,y) ∈ E implies that (y,x) is also in E
